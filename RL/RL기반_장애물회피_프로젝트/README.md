# 강화학습 기반 장애물 회피 자율 주행 시스템 탑재를 위한 이동체 제작

## 프로젝트 진행 단계

1. 이동체를 RC Car로 제작하여 아두이노 제어와 모터 구동 상태 등을 확인
2. Raspberry Pi 5와 카메라 연결 및 테스트
3. 이동체 제어보드인 아두이노와 Raspberry Pi 5 사이의 UART 통신 구현 및 테스트
4. 강화학습 모델 설계 및 구현
5. 이동체에서 RC 부분 제거 후 강화학습 모델 탑재
6. 이동체 주행 환경에서 강화학습 진행
